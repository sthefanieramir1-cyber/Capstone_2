# EmporiUm Sales Analysis — Maryland & New Jersey

Sales analysis project for two assigned EmporiUm territories using Python and Jupyter Notebook. The analysis covers store performance, monthly revenue trends, top customers, and product category insights across 25 stores from January 2022 through December 2025.

## Project File

- **Ramriez_sales_analysis.ipynb** — full analysis notebook covering all six core marketing questions


## Data Files Required

Place all of these in the same folder as the notebook before running:

- `StoreSales.csv`
- `StoreDetail.csv`
- `Products.csv`
- `ProductCategories.csv`
- `customer_list.csv`


## What the Analysis Covers

1. Territory overview — store IDs, locations, and territory managers
2. Monthly total revenue by territory over the full data period
3. Store performance ranking — top performing stores in each territory
4. Top customers — matched using rewards ID and customer list data
5. Product category analysis — transactions and revenue by category per month
6. Marketing recommendation for next quarter based on the full analysis


## How to Run

1. Make sure all CSV files are in the same folder as the notebook
2. Open **Ramriez_sales_analysis.ipynb** in Jupyter
3. Select **Kernel → Restart & Run All**
4. All cells will execute from top to bottom automatically
